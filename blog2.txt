# Interactive Canvas Experiment

Testing out the new custom inline script engine with interactive event hooks. Use the mouse to push the circle, scroll to scale it, or press space to bounce it upwards.

<bs>
// --- 1. Initialization Phase ---
// Variables are declared cleanly; state variables persist across frames.
state x = 200
state y = 150
state speedX = 3
state speedY = 2
state radius = 25
state hitColor = "#00ff00"
state frameCounter = 0

// --- 2. Physics & Logic Phase (Runs Every Frame) ---
tick:
frameCounter = frameCounter + 1

// Move the circle by its current frame speed
x = x + speedX
y = y + speedY

// Check wall collisions (Canvas size is 400x300)
if x > 375:
    speedX = -3
    hitColor = "#ff0000"
else:
    if x < 25:
        speedX = 3
        hitColor = "#0000ff"

if y > 275:
    speedY = -2
    hitColor = "#ffff00"
else:
    if y < 25:
        speedY = 2
        hitColor = "#00ffff"

// Add a subtle wave oscillation to the size over time using math primitives
radius = 25 + sin(frameCounter / 10) * 5

// --- 3. Graphics Rendering Phase ---
draw:
// Wipe the frame clear before redrawing
clear()

// Draw a grid background using lines
color("#222222")
line(0, 150, 400, 150)
line(200, 0, 200, 300)

// Draw the interactive bouncing ball
color(hitColor)
circle(x, y, radius)

// Render dynamic HUD statistics overlays
color("#ffffff")
text("Interactive Sandbox Engaged", 15, 25)
text("Press SPACE to pop up | Click to warp", 15, 45)

// --- 4. Interactive Event Phase Hooks ---
// Warps the circle to the exact coordinates where you left-click
click(mouseX, mouseY):
x = mouseX
y = mouseY
hitColor = "#ffffff"

// Gives the ball an upward thrust when pressing spacebar
key "space":
y = y - 35
speedY = -4

// Increases or decreases velocity thresholds based on wheel scroll delta changes
scroll(dy):
speedX = speedX + dy / 100
</bs>

Everything appears to execute seamlessly in isolated frame spaces. More tests to follow.

# Egg "Shampoo" (Hair Mask) How and Why.

## 3 Egg Yolks
Egg yolks are rich in fats and phospholipids, like lecithin. Lecithin is a natural emulsifier that spreads oils evenly over the hair, and the lipids leave a smooth conditioned feel.

## Coconut Oil 
Coconut Oil is of the few oils that have actually been shown to penetrate the shaft of the hair to an extent. Very high in lauric acid.

## Raw Honey
Raw honey attracts and holds water. Increases softness when added in small amounts and properly incorporated into the mix, too much will make your hair feel sticky.

## Corn Starch 
Corn Starch changes the texture of the mask, good for making the hair mask feel professionally made. This can be omitted.

## Salt 
Salt changes the texture of the hair so that it doesn't feel greasy, it also helps prevent spoilage by a little bit.

## Essential Oils
Essential oils are added just for fragrance. Dilute the essential oils so that your skin doesn't get irritated. I myself use extracts.

I've seen many recipes for egg hair masks but I've not seen a recipe as well-planned as mine. For EVERYTHING on this blog I will think about these quotes from Terry Davis: "How can I make this not Niggerlicious?" and "An idiot admires complexity, a genius admires simplicity."

I've noticed that many people that make hair masks with egg yolks just mix everything together without much forethought. Do not do that. When mixing this together, you should first mix together the egg yolks with the coconut oil. Add in the coconut oil little by little until the egg yolk and coconut oil mixture begins to have a creamy texture much like aioli or a light mayonnaise.

After that, stir in a little bit of raw honey (Eyeball it. Two teaspoons seems to work the best for me.) and then after that stir in your corn starch. The corn starch should be added little by little, you'll end up getting chunks of corn starch if you do not slowly incorporate it.
