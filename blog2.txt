# 3D Physical Space Experiment

Simulating a ball with gravity and velocity vector math inside a virtual 3D bounding box. Use your mouse to click and drag to orbit the camera, or press W/S to move the camera forward and backward.

<bs>
// --- 1. Initialization Phase ---
state x = 0
state y = 0
state z = 0
state vx = 2
state vy = 1
state vz = 3
state radius = 15

// Environmental constants
state gravity = 0.2
state bounce = -0.85
state boxSize = 120

// Camera control parameters
state camAngleX = 0.5
state camAngleY = 0.3
state camZoom = 250

// Mouse interaction tracking
state isDragging = 0
state lastMouseX = 0
state lastMouseY = 0

tick:
// --- 2. Physics Logic Phase ---
// Apply gravity along the Y axis
vy = vy + gravity

// Update positions via velocity increments
x = x + vx
y = y + vy
z = z + vz

// Bounding box collisions (Centered at 0,0,0 with bounds at +/- boxSize)
if abs(x) > boxSize:
    vx = vx * bounce
    x = min(max(x, -boxSize), boxSize)

if abs(y) > boxSize:
    vy = vy * bounce
    y = min(max(y, -boxSize), boxSize)

if abs(z) > boxSize:
    vz = vz * bounce
    z = min(max(z, -boxSize), boxSize)

draw:
// --- 3. Graphics Rendering Phase ---
clear()

// Pre-calculate rotation trigonometric primitives for the camera matrix
state cx = cos(camAngleX)
state sx = sin(camAngleX)
state cy = cos(camAngleY)
state sy = sin(camAngleY)

// Helper logic to project 3D coordinates (X, Y, Z) into 2D Screen Space (PX, PY)
// Translates and projects relative to the center of the canvas window (200, 150)
// We use temporary step evaluations since equations run linearly:

// --- Render Bounding Wireframe Structure ---
color("#333333")
// We can approximate a basic perspective viewport by drawing cross lines at the center point
line(100, 50, 300, 50)
line(100, 250, 300, 250)
line(100, 50, 100, 250)
line(300, 50, 300, 250)

// --- Transform Ball Coordinates into View Space ---
state rotX = x * cx - z * sx
state rotZ = x * sx + z * cx
state rotY = y * cy - rotZ * sy
state finalZ = y * sy + rotZ * cy + camZoom

// Project to 2D screen surface if it sits safely ahead of the near plane clipping point
if finalZ > 50:
    state screenX = 200 + (rotX * 200) / finalZ
    state screenY = 150 + (rotY * 200) / finalZ
    state sizePerspective = (radius * 200) / finalZ

    // Render shadow projection on the floor grid plane
    color("#112211")
    circle(screenX, 230, sizePerspective + 2)

    // Draw the main interactive physical sphere object
    color("#00ff66")
    circle(screenX, screenY, sizePerspective)

// Render active telemetry overlays on screen margins
color("#ffffff")
text("Camera Matrix Active", 15, 25)
text("Drag Mouse to Orbit | Press W/S to Zoom", 15, 45)

// --- 4. Interactive Event Phase Hooks ---
click(mouseX, mouseY):
isDragging = 1
lastMouseX = mouseX
lastMouseY = mouseY

// Track dragging mechanics outside standard clicks via update ticks if required,
// or alter view matrices incrementally whenever keys are held.
key "w":
camZoom = camZoom - 10

key "s":
camZoom = camZoom + 10

key "a":
camAngleX = camAngleX - 0.05

key "d":
camAngleX = camAngleX + 0.05
</bs>

# Egg "Shampoo" (Hair Mask) How and Why.

## 3 Egg Yolks
Egg yolks are rich in fats and phospholipids, like lecithin. Lecithin is a natural emulsifier that spreads oils evenly over the hair, and the lipids leave a smooth conditioned feel.

## Coconut Oil 
Coconut Oil is of the few oils that have actually been shown to penetrate the shaft of the hair to an extent. Very high in lauric acid.

## Raw Honey
Raw honey attracts and holds water. Increases softness when added in small amounts and properly incorporated into the mix, too much will make your hair feel sticky.

## Corn Starch 
Corn Starch changes the texture of the mask, good for making the hair mask feel professionally made. This can be omitted.

## Salt 
Salt changes the texture of the hair so that it doesn't feel greasy, it also helps prevent spoilage by a little bit.

## Essential Oils
Essential oils are added just for fragrance. Dilute the essential oils so that your skin doesn't get irritated. I myself use extracts.

I've seen many recipes for egg hair masks but I've not seen a recipe as well-planned as mine. For EVERYTHING on this blog I will think about these quotes from Terry Davis: "How can I make this not Niggerlicious?" and "An idiot admires complexity, a genius admires simplicity."

I've noticed that many people that make hair masks with egg yolks just mix everything together without much forethought. Do not do that. When mixing this together, you should first mix together the egg yolks with the coconut oil. Add in the coconut oil little by little until the egg yolk and coconut oil mixture begins to have a creamy texture much like aioli or a light mayonnaise.

After that, stir in a little bit of raw honey (Eyeball it. Two teaspoons seems to work the best for me.) and then after that stir in your corn starch. The corn starch should be added little by little, you'll end up getting chunks of corn starch if you do not slowly incorporate it.
